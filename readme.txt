[color=#572500][b]Features[/b][/color]
Add a build-in search engine to your forum.

[color=#572500][b]Installation[/b][/color]
Any previous versions of this mod [b]MUST[/b] be uninstalled [b]BEFORE[/b] installing this version.

[color=#572500][b]Usage[/b][/color]
[list]
[li]After install create your own Google Custom Search Engine: [url=http://www.google.com/cse/]http://www.google.com/cse/[/url][/li]
[li]Set up modification settings. Copy engine ID or HTML code and switch on modification (Admin -> Configuration -> Modification Settings).[/li]
[/list]
